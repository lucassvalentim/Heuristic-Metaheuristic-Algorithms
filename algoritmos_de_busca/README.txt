Exercícios de Algoritmos de Busca

Este repositório contém exercícios separados em duas pastas, cada uma abordando um tipo de algoritmo de busca.

Estrutura do Repositório

1. Pasta busca exaustiva:
Esta pasta contém todos os exercícios relacionados ao tema de busca exaustiva. Para executar os programas nesta pasta, basta compilar o código com o compilador g++ e executá-los. 

Observação:
Para o exercicio2b.cpp, é necessário utilizar um arquivo de entrada chamado exercicio2b.txt. Execute-o da seguinte maneira:

./exercicio2b < exercicio2b.txt

2. Pasta busca local:
Esta pasta contém exercícios sobre busca local, separados em duas subpastas:

exercicio3_mochila (problema da mochila)
exercicio4_pcv (problema do caixeiro-viajante)

Cada um dos exercícios desta pasta também requer a leitura de arquivos de entrada. O processo para compilar e rodar cada um deles é semelhante:

Compile o código:
g++ nome_do_arquivo.cpp -o nome_do_executavel

Execute o programa com o arquivo de entrada necessário:

./nome_do_executavel < nome_do_arquivo_de_entrada.txt
